# Changelog

All notable changes to X-Agent will be documented in this file.

## [3.0.0] - 2026-03-25

### Added
- 🎭 Multi-niche support with 7 built-in voice files
- 🔥 Deep integration with last30days-skill for 8+ platform monitoring
- 📊 Compound scoring system (Relevance/Velocity/Authority/Convergence)
- 🔒 Anti-ban mechanisms (random delays, content variants, daily limits)
- 📱 Complete Telegram Bot with 15+ commands and inline buttons
- 🧪 Unit tests for all core modules
- 📚 Bilingual documentation (English/Chinese)

### Changed
- Niche voice files changed from `.txt` to `.md` format
- Research module now uses last30days-skill for deep research
- Generator supports media suggestions and optimal posting times
- OpenClaw bridge enhanced with stealth mode

### Fixed
- Database connection stability
- LLM router error handling
- Telegram Bot button callbacks

## [2.0.0] - 2026-03-24

### Added
- Multi-LLM router support (Claude/OpenAI/Groq/Gemini/OpenRouter/NVIDIA/Ollama)
- Supabase database integration
- Daily review and strategy generation
- Trending topic monitoring

### Changed
- Modular architecture with separate modules
- Configuration management improved

## [1.0.0] - 2026-03-23

### Added
- Basic Twitter content generation
- Simple scoring system
- Telegram Bot prototype
