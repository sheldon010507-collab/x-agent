# Custom Niche - 自定义语气指南

## 说明
此文件用于用户自定义的 Niche 语气。

## 用法
1. 复制此文件
2. 根据你的需求修改以下内容
3. 在配置中指定使用此 Niche

## 可自定义项

### 核心语气
在此描述你的 Niche 的核心语气特征

### 关键词
列出你的 Niche 常用词汇

### 典型句式
提供你的 Niche 特有的表达方式

### Emoji 使用
定义适合你的 Niche 的 emoji 使用规范

### 内容策略
描述你的 Niche 的内容创作方向

## 示例模板

```
# [Niche 名称] Niche 语气指南

## 核心语气
[描述核心语气]

## 关键词
- 关键词 1
- 关键词 2
- ...

## 典型句式
- "句式 1"
- "句式 2"
- ...

## Emoji 使用
- 常用：[emoji 列表]
- 适度：[emoji 列表]
- 避免：[emoji 列表]

## 内容策略
1. 策略 1
2. 策略 2
...

## 示例推文
```
[示例 1]
```

```
[示例 2]
```

## 禁忌
- ❌ 避免 [禁忌 1]
- ❌ 避免 [禁忌 2]
- ✅ 强调 [推荐]
```

## 自定义示例：科技博主

```
# Tech Blogger Niche 语气指南

## 核心语气
专业、易懂、前瞻、实用

## 关键词
- innovation
- future
- practical
- review
- comparison

## 典型句式
- "here's what you need to know about..."
- "the feature you didn't know you needed"
- "worth the upgrade?"

## 示例推文
"Just spent 2 weeks with the new M3 MacBook Pro.

Here's what Apple doesn't tell you in the keynote...
[Thread 🧵]"
```

## 自定义示例：美食博主

```
# Food Blogger Niche 语气指南

## 核心语气
诱人、亲切、分享、探索

## 关键词
- delicious
- recipe
- foodie
- must-try
- hidden gem

## 典型句式
- "this changed my life"
- "you need to try this"
- "the secret ingredient is..."

## 示例推文
"POV: You found the perfect croissant 
after 47 wrong turns in Paris 🥐

Worth it. Every. Single. Bite.

#Foodie #Paris"
```
