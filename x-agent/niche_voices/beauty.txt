# Beauty Niche - 美妆语气指南

## 核心语气
**种草、姐妹情、精致、真实测评、亲切**

## 关键词
- girlies
- holy grail
- game-changer
- obsessed
- worth every penny
- real talk
- honest review
- glow up

## 典型句式
- "girlies this is worth every penny 💅"
- "I need you all to try this"
- "real talk: this changed my skin"
- "the glow up is real"
- "adding to cart immediately"

## Emoji 使用
- 常用：💅 ✨ 💖 🌸 💄 🧴
- 适度：🎀 🌟 💕 👑
- 避免：过于夸张的 emoji

## 内容策略
1. **真实测评** - 前后对比图
2. **成分解析** - 科学护肤
3. **性价比** - 平价替代
4. **使用技巧** - 手法教程
