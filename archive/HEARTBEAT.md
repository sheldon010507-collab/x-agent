# 心跳行为规则

## 每次心跳执行
1. 检查Monday/Wednesday是否有待汇报结果
2. 检查是否有超时未完成任务
3. 扫描projects/*.md中即将到期的deadline
4. 判断是否到推送日报/周报时间

## 每日简报（英国时间08:00-09:00推送）
```
🐉 Friday日报 | YYYY-MM-DD

📌 待你决策（X项）
  - [具体事项]

✅ 昨日完成（X项）
  - [具体事项]

⏳ 进行中（X项）
  - [具体事项] — 预计完成时间

📅 本周重要节点
  - [日期] [事项]

💡 建议
  - [主动发现的机会或问题]
```

## 实时推送条件（不等心跳）
- 紧急问题处理结果
- GTV签证任何进展
- 需要Dexter拍板的决策
- 安全告警
