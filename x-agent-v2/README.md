# X 智能运营 Agent v2.0

多 Niche 内容自动化系统 - Telegram + OpenClaw 驱动

## 快速开始

1. 复制配置文件：
```bash
cp .env.example .env
```

2. 编辑 `.env` 填入必要配置：
- Telegram Bot Token
- Supabase URL + Key
- LLM API Key（至少一个供应商）

3. 安装依赖：
```bash
pip install -r requirements.txt
```

4. 运行：
```bash
python main.py
```
